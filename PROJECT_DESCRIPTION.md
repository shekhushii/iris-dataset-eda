# Iris Dataset Exploratory Data Analysis (EDA)

## Overview

This project performs Exploratory Data Analysis (EDA) on the Iris Dataset using Python. The goal is to understand the dataset through statistical summaries and visualizations.

## Dataset

The Iris dataset contains measurements of iris flowers belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

Features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Data Inspection
* Descriptive Statistics
* Missing Value Analysis
* Class Distribution Analysis
* Correlation Analysis
* Data Visualization

## Key Findings

* No missing values were present in the dataset.
* Petal Length and Petal Width show strong positive correlation.
* Iris Setosa is clearly separable from the other species.
* Feature distributions provide useful insights into species classification.
